# SQL-Portfolio-Projects
Operations &amp; Business Analytics
Project1
You are a Quality Analyst at a streaming platform (like Prime Video). Each week, content metadata records are submitted for quality review. Your manager needs a dashboard that shows how many records passed or failed QA, broken down by content type and region — to track operational health and identify problem areas

Project 2
You are a Program Operations Analyst tracking operational SLAs. Each incoming work ticket must be resolved within a defined SLA window (48 hours). You need to report: how many tickets breached SLA, average resolution time by team, and which ticket categories have the worst compliance rate — to drive process improvement initiatives.
